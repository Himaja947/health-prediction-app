# Health Prediction Application

## Features

- Create Patient Records
- View Records
- Update Records
- Delete Records
- Input Validation
- SQLite Database Storage
- AI-powered Health Prediction using Gemini API
- CSV Export
- Search Functionality

## Tech Stack

- Python
- Streamlit
- SQLite
- Gemini AI API

## Run

pip install -r requirements.txt

streamlit run app.py
